Ainda Sobre as Tags:

(VOLTAR)

Os títulos são usados a partir dos H , indo do h1 até o h6. Tags podem ser INLINE ou INBLOCK, que significam que podem ocupar um bloco, uma linha inteira ou somente o conteúdo dela.

O p é uma tag INBLOCK e o span é uma tag INLINE
